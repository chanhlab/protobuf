# Protobuf

## Guides

### Create a new proto

```
touch chanhlab/sample/v1/sample.proto
```

### Linter

```
make lint
```

### Generate protobuf

```
make generate
```

### Pull

```
GOPRIVATE=github.com/chanhlab go get -u github.com/chanhlab/protobuf
```
