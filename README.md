# Protobuf

## Guides

### Create a new proto

```
prototool create chanhlab/event/v1/event.proto
```

### Generate protobuf

```
prototool generate
// or
make generate
```

### Pull

```
GOPRIVATE=github.com/chanhlab go get -u github.com/chanhlab/protobuf
```
